# 프로젝트 소개
**현재 문제점**
- 교육과정 공지용 채팅방에 수업시간 채팅도 함께 올라오는 상황
- 이미 3개의 카카오톡 톡방 존재
- 줌 채팅의 휘발성
- 다소 불편한 인스타그램 pc버전 UI/UX

**DARK CIRCLE 채팅프로그램의 필요성**
- 카카오톡과 분리된 독자적 채팅앱
- 사용자 편의성을 고려한 기능
- 보안성을 고려한 채팅앱
- 대규모 서비스 확장 고려한 개발
- 성능 최적화
